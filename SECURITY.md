# Security Policy / 安全策略

Ergouzi Switch 群友版基于 CC Switch 改写和重新分发。本仓库只维护 Ergouzi Switch 群友版自身的安全问题。

Ergouzi Switch is a community distribution based on CC Switch. This repository only handles security issues for the Ergouzi Switch distribution.

## Supported Versions / 支持的版本

Only the latest Ergouzi Switch release receives community security updates.

仅最新版 Ergouzi Switch 群友版会接收社区安全修复。

| Version / 版本 | Supported / 是否支持 |
| -------------- | -------------------- |
| Latest 3.x     | Yes / 是             |
| < 3.0          | No / 否              |

## Reporting a Vulnerability / 报告漏洞

Please do not report security vulnerabilities through public GitHub issues.

请不要通过公开 GitHub Issue 报告安全漏洞。

Report privately through this repository's GitHub Security Advisories:

请通过本仓库的 GitHub Security Advisories 私下报告：

https://github.com/suboxiu88-coder/ergouzi-switch/security/advisories/new

When reporting, please include:

报告时请包含：

- Vulnerability description / 漏洞描述
- Steps to reproduce / 复现步骤
- Potential impact / 潜在影响
- Affected versions / 受影响版本

## Security Updates / 安全更新

Security fixes are released through this repository's GitHub Releases:

安全修复会通过本仓库 GitHub Releases 发布：

https://github.com/suboxiu88-coder/ergouzi-switch/releases
